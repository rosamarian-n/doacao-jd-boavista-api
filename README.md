# Doação JD Boavista API ♻️

API para doação de itens, conectando quem quer doar com quem precisa no JD Boavista.

## 🚀 Tecnologias

- Python + FastAPI
- SQLAlchemy (ORM)
- SQLite
- Docker
- Swagger/OpenAPI

## 📋 Funcionalidades

- ✅ Cadastro de usuários
- ✅ Listagem de usuários
- ✅ Validação de dados
- ✅ Banco de dados integrado
- ✅ Documentação automática
## 🏗️ Como executar

### Com Docker (recomendado)

```bash
# Clone o repositório
git clone https://github.com/rosamarian-n/doacao-jd-boavista-api.git

# Entre na pasta
cd doacao-jd-boavista-api

# Execute com Docker
docker-compose up --build
